# BRL to USD Converter 

Aplicação web simples para conversão de Real (BRL) para Dólar (USD), desenvolvida com foco em prática de HTML, CSS e principalmente JavaScript.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- API de cotação em tempo real

## Funcionalidades

- Conversão de BRL para USD
- Exibição da cotação atual
- Atualização dinâmica dos valores
- Interface simples e responsiva
